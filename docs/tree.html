Project structure of: james4ever0/lazero

  • lazero Efficient AI toolkit with network utilities, tests and installation.
    • install.sh Uninstall, clean, reinstall lazero.
    • lazero Efficient network utility with Lazero, async GET and downloading.
      • __main__.py Search, create indexes, show FFmpeg filters with argparse utility.
      • filesystem Efficient I/O, temporary manager, conflict-free imports
        • __init__.py Importing all modules may cause conflicts or duplicates.
        • env.py User home directory function
        • io.py Efficient file I/O with multiple formats support
        • temp.py Temporary file/dir manager with clean-up and existence check
      • network Network module directory for Lazero, featuring efficient downloading and async GET.
        • __init__.py Lazero network modules import.
        • asyncio.py Asynchronous aiohttp GET function with concurrency.
        • checker.py Python network checker, waits for server response or returns False.
        • downloader.py Efficient file downloader using lazero network.
        • progressbar Network progress bar client and server.
          • client.py Network progress bar client with potential errors.
          • server.py Progress bar server with FastAPI, routes for open, update, close.
        • proxy
          • clash.py Test and sort proxies for Clash.
      • program Program module imports, functions, and utilities.
        • __init__.py Imports lazero.program modules.
        • functools.py Temporary directories, exception handling, decorators.
        • subprocess.py Runs command, returns output as JSON or error.
        • threading.py Threading module, define startThread and asyncThread functions.
      • search Directory contains search scripts and interfaces for various libraries.
        • api.py Python script for file path retrieval, database storage and indexing.
        • index.py Search indexing script for TinyDB database.
        • postprocessing.py Post-processing Python script for search results with highlighting and logging.
        • preprocessing.py Preprocesses text, formats lines, performs windowed convolution.
        • search.py Searches, retrieves, ranks, and prepares data.
        • terminal_interface.py Searchable terminal interface with dynamic height and keyboard support.
        • txtai Indexes and searches using TxtAI library.
          • index.py Indexes directory using TxtAI library.
          • model.py Embeddings model path initialization for sentence transformer.
          • search.py Text search with TxtAI index and result limits.
        • whoosh Search and index files with Whoosh Python library.
          • index.py Indexes files using Whoosh search library.
          • model.py Defines schema, creates index for Whoosh search.
          • search.py Whoosh Python search with optional filters and weighted fields.
      • utils Data processing, JSON manipulation, math library, importers, inspectors, logger, tools.
        • __init__.py Lazero utils package import utility functions.
        • importers.py Sets OpenCV path for Python3.9, checks and inserts .so file.
        • inspectors.py Inspects objects, prints variables or iterates keys.
        • json.py Functions for JSON manipulation and updates.
        • logger.py Utility logger functions: "sprint" and "traceError".
        • mathlib.py Python math library for data processing, merging intervals.
        • tools.py Manipulate data structures, custom iterators, handle nested generators.
    • README.md Lazero: AI toolkit for unlimited tasks.
    • setup.py Python package setup: lazero, alpha, education, multi-platform.
    • signal_2.sh Exits script with status code 2.
    • test_exit.sh Test script for exit code 2 in signal_2.sh
    • tests Test various functionalities and interactions.
      • test_async_concurrentGet_pickledFunction.py Error awaiting ClientResponse object in async concurrent GET.
      • test_commons.py Updated PYTHONPATH, disabled proxy, imported modules.
      • test_noqlite.py Test key-value storage and retrieval in Noqlite database.
      • test_search.py Test Python search functionality.
      • test_search_index.py Test search index functionality.
      • test_search_index_notable.sh Lazero indexes notes.
      • test_textual_panel.py Test text panel creation with red text, right-aligned title, left-aligned subtitle, green styling.
      • test_tiny_db.py TinyDB test: simple No-SQL database interactions.
      • test_view_db_search.py Testing 100 keys' values, database search.
      • test_walk_dirs.py Tests directory traversal and printing file paths.
